# Google Search Console Setup

Use this checklist after the GitHub Pages site is live.

## Property

Create a URL-prefix property:

```text
https://zgnme.github.io/pureautolike/
```

## Sitemap

Submit this sitemap:

```text
https://zgnme.github.io/pureautolike/sitemap.xml
```

## Request Indexing

Open URL Inspection and inspect:

```text
https://zgnme.github.io/pureautolike/
```

Then click **Request indexing**.

## Verification Note

Google Search Console verification must be completed from the owner's Google
account. If Google gives an HTML verification file or meta tag, add it to
`site/`, deploy GitHub Pages, then verify again.
