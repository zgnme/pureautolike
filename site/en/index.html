Pure Web beta extension

Pure auto liker for the Pure Web feed.

PureAutoLike is a Chrome extension for Pure Web users who want faster visible-feed likes, accessible photo opening, Telegram alerts, session timers, and local Markdown export without a separate desktop automation app.

Install from Chrome Web Store View source on GitHub

What it does

A small extension for a specific Pure workflow.

Pure auto liker

Clicks visible like controls in the open Pure Web tab and avoids re-processing the same visible profile during the current session.

Hidden photo opener

Adds a page control for Pure photo placeholders when the active web session already has access to those photos.

Telegram notifications

Optionally sends alerts for new Pure matches, likes, and messages after you add your own Telegram bot token and chat id.

Local Markdown export

Stores visible profile status, age, and description locally only when capture mode is enabled, then exports them manually.

Search intent

Built for people searching for a Pure app auto liker, not a generic autoclicker.
