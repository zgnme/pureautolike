Why an extension

A browser extension stays closer to the page than a fixed-coordinate autoclicker.

Browser support

The public one-click install path is Chrome Web Store. Chromium browsers are close, but public distribution is focused on Chrome.

Privacy position

Settings stay in extension storage, Telegram is manual, and the extension has no third-party analytics.

What PureAutoLike does

  • Auto-like for visible profiles in an open Pure Web tab.
  • Opening only photos available to the current Pure session.
  • Optional Telegram alerts for matches, likes, and messages.
  • Local Markdown export of visible descriptions when enabled manually.
