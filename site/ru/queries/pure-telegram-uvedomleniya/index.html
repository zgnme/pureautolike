Бесплатная beta 2026: Telegram-уведомления для Pure Web вместе с автолайком.

PureAutoLike может присылать Telegram-события, когда в Pure появляются матчи, лайки или сообщения. Функция выключена по умолчанию и включается вручную.

Что отправляется

События о матчах, лайках и сообщениях, если пользователь сам ввел bot token и chat id.

Почему это важно

Автолайк убирает рутину, а Telegram помогает быстрее заметить событие, на которое стоит реагировать.

Граница приватности

Token и chat id хранятся в extension storage и используются только для Telegram Bot API при включенной функции.

Что делает PureAutoLike

  • Автолайк видимых анкет в открытой вкладке Pure Web.
  • Открытие только доступных фото текущей Pure-сессии.
  • Опциональные Telegram-уведомления о матчах, лайках и сообщениях.
  • Локальный Markdown-экспорт видимых описаний при ручном включении.
